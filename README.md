# Apache Iceberg Lakehouse Interoperability Examples

This repository provides practical examples supporting the whitepaper:

"Evaluating Apache Iceberg for Interoperable Lakehouse Architectures in Azure-Based Databricks and Snowflake Environments"

Contents include:

• Iceberg table creation in Databricks  
• PySpark ingestion examples  
• Snapshot and time travel queries  
• Snowflake Iceberg table configuration  
• Table optimization and compaction strategies  

These examples demonstrate how Apache Iceberg can enable cross-platform lakehouse architectures across Databricks and Snowflake using Azure Data Lake Storage.
